# workflow-generator

## About the project

A Vue.Js single page app that allow the user to generate workflow by adding nodes and connecting them

## live demo

CodeSandBox: [Workflow Generator App]()

## Setup instructions

1. Install npm packages

   ```bash
   npm i
   ```

2. Run the app

   ```bash
   npm run serve
   ```

## Testing

```bash
npm run test
```
